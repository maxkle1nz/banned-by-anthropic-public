# Contributing

This repository is the public institutional archive of the BANNED BY ANTHROPIC campaign.

## Good contributions

- factual corrections
- wording fixes that improve clarity without changing meaning
- documentation improvements
- schema/docs improvements for public snapshots

## Not accepted here

- private case submissions
- raw dumps of personal data
- operational security details
- moderation internals
- harassment or speculation

## For affected people

If you want to register a case, use the official site:

- https://bannedbyanthropic.com/
