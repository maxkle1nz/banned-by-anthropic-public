# Press Kit

## Short description

BANNED BY ANTHROPIC is a public archive of documented cases from builders and users affected by Anthropic enforcement actions.

## What this repository provides

- campaign framing
- public methodology
- publication boundaries
- official updates
- public reference material

## What this repository does not provide

- raw private submission dumps
- operational internals
- admin access
- private participant data

## Recommended reference links

- Official site: https://bannedbyanthropic.com/
- Public repository: https://github.com/maxkle1nz/banned-by-anthropic-public
