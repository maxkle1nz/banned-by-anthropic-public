# FAQ

## What is BANNED BY ANTHROPIC?

A public archive of documented cases from builders and users affected by Anthropic enforcement actions.

## Is this repository the submission system?

No. This repository is the public institutional archive of the campaign. Case submission happens through the official site.

## Can private cases still count?

Yes. A case may contribute to aggregate reporting without becoming fully public.

## Does this repository publish raw data?

No. It should publish only curated public outputs, methodology, policies, and official updates.

## Can I submit a correction?

Yes. Use the factual correction issue template in this repository.

## Can I contact the campaign for press or research?

Yes. Use the media inquiry issue template in this repository.
